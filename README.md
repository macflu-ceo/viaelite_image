# VIA ELITE 상품카드 생성기

비아엘리떼 컨시어지가 명품 상품을 영업할 때 사용하는 카드 이미지 자동 생성 도구입니다.
브랜드명·가격·상품 이미지만 입력하면 8가지 디자인 템플릿으로 인스타·카카오톡용 카드를 즉시 생성합니다.

> **The Gateway to Global Luxury Distribution**
> J Primo International · VIA ELITE

---

## 주요 기능

- **단일형 8개 템플릿** — 가격 강조, 할인율 영웅형, 특가 카피, 시즌 한정, 럭셔리 다크, 풀이미지 오버레이, 백화점 비교, 미니멀 시그니처
- **리스트형** — 여러 상품을 한 화면에 큐레이션
- **구글 시트 기반 컨시어지 인증** — 시트에서 회원 관리, ID/비밀번호로 로그인
- **세션 자동 만료** — 탭/창 닫으면 자동 로그아웃
- **PNG 2x 고해상도 다운로드** — 인스타·카톡 공유 최적화
- **마지막 로그인 기록** — 시트에 자동 기록되어 활동 모니터링 가능

## 시작하기

### 사용자(컨시어지)
`index.html`을 브라우저에서 열면 됩니다. 본사에서 발급받은 ID/비밀번호로 로그인 후 사용.

### 본사 관리자 (최초 설정)
1. `docs/setup-guide.html`을 브라우저에서 열기
2. 가이드의 1~6단계를 따라 약 15~20분간 설정:
   - 구글 시트 생성 및 컨시어지 명단 입력
   - Apps Script 코드(`server/viaelite_auth.gs`) 등록 및 배포
   - 발급받은 URL을 `index.html`의 `APPS_SCRIPT_URL`에 입력
3. 테스트 로그인으로 검증

## 컨시어지 명단 관리 (구글 시트)

| ID | 비밀번호 | 이름 | 활성 | 레벨 | 소속 | 가입일 | 마지막 로그인 | 로그인 횟수 |
|---|---|---|---|---|---|---|---|---|
| kim001 | vip2026 | 김민지 | TRUE | 시니어 | 강남지점 | 2026-01-15 | (자동) | (자동) |

- **활성** 컬럼을 `FALSE`로 변경하면 즉시 로그인 차단
- **마지막 로그인**, **로그인 횟수**는 시스템이 자동 기록
- 새 컨시어지 추가는 시트에 행만 추가하면 즉시 적용 (HTML 수정 불필요)

## 보안 정책

이 시스템은 **"비공식 내부 도구" 수준의 보안**으로 설계되었습니다. 컨시어지 명단 관리·내부 도구 접근 통제 용도로 적합하지만, 일반 회원가입/결제 서비스 수준의 보안은 아닙니다.

**중요**

- 이 저장소는 **비공개(private) 저장소로 유지**해야 합니다 — `index.html`에 Apps Script URL이 포함되어 있어 공개되면 외부에서 로그인 시도 가능
- 구글 시트의 비밀번호는 평문 저장 — **시트 공유 권한을 본사 인원에게만 한정**
- 컨시어지에게는 시트 권한을 부여하지 마세요 (HTML 파일만 배포)
- 컨시어지 100명 이내 규모에 적합. 그 이상이면 정식 백엔드로 이전 권장

## 운영 가이드

### 컨시어지 추가
구글 시트의 빈 행에 정보 입력 → 즉시 로그인 가능.

### 컨시어지 차단 (퇴사·자격박탈)
시트의 해당 행 **활성** 컬럼을 `FALSE`로 변경 → 다음 로그인 시도부터 차단.

### 비밀번호 변경
시트에서 직접 **비밀번호** 컬럼 수정.

### 활동 모니터링
시트의 **마지막 로그인**, **로그인 횟수** 컬럼으로 누가 활발히 쓰는지 한눈에 파악.

## 향후 확장 방향

현재는 컨시어지 수십 명 규모의 베타 테스트 단계 도구입니다. 다음 단계로 확장 시 고려할 사항:

- **외부 호스팅** — Vercel/Netlify에 배포하여 컨시어지가 URL로 접속
- **본사 관리자 대시보드** — 활동 통계, 카드 생성 횟수, 인기 브랜드 분석
- **정식 회원 시스템** — 회원가입 페이지, 비밀번호 재설정, 권한 관리
- **카드 템플릿 추가** — 컨시어지 본인 사진/연락처 자동 삽입, QR 코드, 시즌 캠페인용
- **공유 추적** — 카드별 공유 URL, 클릭/뷰 통계
